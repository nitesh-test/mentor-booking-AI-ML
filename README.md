# Mentor Booking AI/ML

Initial scaffolding for the Mentor Booking AI/ML project.

## Setup

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run tests:

```bash
pytest
```
